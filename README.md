# voicenav

VoiceNav is an innovative web application that revolutionizes how users interact with websites and documents through natural voice commands. Built with modern web technologies, it provides hands-free navigation and control, making web browsing more accessible and intuitive.

Features
Voice-Activated Navigation - Navigate through web pages using simple voice commands

Multi-accent Support - Supports different English accent voice commands and outputs

speed adjusting

Real-time Voice Recognition - Instant response to user commands with visual feedback

Smart Command Processing - Intelligent interpretation of natural language commands

Accessibility Focus - Designed to assist users with blindnness,mobility impairments or temporary disabilities,reading disabilities 

Customizable Commands - Easy-to-configure voice command mappings

Page Navigation - Navigate between different sections/pages

Scroll Control - Scroll up, down, top, and bottom

Click Simulation - Click on buttons, links, and elements by voice

Form Interaction - Fill forms using voice input

Browser Controls - Back, forward, refresh, and new tab commands
